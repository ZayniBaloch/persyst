Never lose context again.

A local-first, zero-egress MCP memory layer. Give your AI coding agents permanent memory, hybrid semantic search, and knowledge graph capabilities across sessions.

$ npx persyst-mcp

Instantly start the memory server. Zero API keys required.

Why Persyst?

Hybrid Search Engine

Combines FTS5 exact keyword matching with HuggingFace local semantic embeddings for perfect recall.

Git Ingestion

Turn your entire repository history into a searchable memory bank. Agents know why you wrote that code.

Knowledge Graph

Built-in entity and edge tables allow agents to traverse relationships between files, concepts, and people.

100% Local & Private

Everything runs entirely on your machine via SQLite and local transformers. Zero cloud dependencies. Zero egress.
