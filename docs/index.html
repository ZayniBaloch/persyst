Context that persists.

Engineered for Autonomous Agents

Hybrid Search Engine

Combines FTS5 exact keyword matching with HuggingFace local semantic embeddings for perfect recall across memory banks.

Git History Ingestion

Turn your entire repository history into a searchable memory bank. Agents can instantly retrieve historical technical decisions.

Knowledge Graph

Built-in entity and edge tables allow agents to traverse relational connections between files, abstract concepts, and authors.

Zero Egress. 100% Local.

Operates entirely on your machine via SQLite and local transformer models. No cloud dependencies, no API keys required.

Seamless Integration

Persyst implements the Model Context Protocol (MCP). Any compliant agent can immediately utilize its tools over standard stdio streams.

  • search_memories
  • ingest_git_commits
  • link_entity_memory
  • get_important_memories
aider --model claude-3.5-sonnet
User> Why did we choose Postgres?
System: Calling tool search_memories({"query":"Postgres"})
Agent> Based on a memory from last month, we chose Postgres for its superior JSONB support.
User> Remember to use Hooks for new components.
System: Calling tool add_memory({"content":"Use Hooks for components"})
Agent> Stored in memory successfully.
